# EpiqCodingInterview
Installations:\
  NPM ->
    Windows:
      https://nodejs.org/en/download/ \
     -> Linux (Ubuntu, Debian):
      $ sudo apt install npm
      OR
      $ sudo apt install npm nodejs\
    -> Linux (Arch Linux, Manjaro): 
      $ sudo pacman -S npm\
 React ->
    npm install react\
 How to run:\
    Once cloned from repo:\
          -> cd to census-project\
          -> run the command npm install\
          -> run the command npm start
